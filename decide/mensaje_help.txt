-> /login <nombre_de_usuario> <contraseña> - Inicia sesión para poder votar

-> /logout - Cierra sesión

-> /vote <id_usuario> <id_votacion> <id_opcion> - Para votar en una votación si se tiene permiso

-> /votaciones - Muestra un listado con todas las votaciones y sus id

-> /votacion <id_votacion> - Muestra información detallada de la votación

-> /options <id_votacion> - Muestra opciones de votacion

-> /censos - Muestra un listado con todos los censos

-> /usuarios - Muestra un listado con todos los usuarios
